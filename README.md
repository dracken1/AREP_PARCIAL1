# Parcial 1 AREP
  
  
# Getting Started
## Install with

```
git clone https://github.com/dracken1/AREP_PARCIAL1.git
```

## Run with

Dentro de la carpeta AREP_LAB1, inice cmd e intrduzca los siguientes comandos:

```
mvn clean

mvn compile

mvn test
```

o tabmien:

```
mvn package
```

# Build with
El proyecto se desarrollo usando el lenguaje Java y la herramienta Intellij.
